# project2PSF

Creating an ecommerce site with symfony framework

### Launch app

```
make
```

### Build project

```
make build
```

### Install all dependencies

```
 make install
```

### Up project

```
make up
```

### Down container

```
make down
```

### Executing command inside the container

```
make exec
```

### Database

#### execute migration

```
make migration
```

### Asset

#### compile asset

```
make asset
```
